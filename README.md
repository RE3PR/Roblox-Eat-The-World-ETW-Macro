# 🍽️ ETW Macro (Eat The World – Roblox)

This is an **open-source macro project** designed for automation in *Eat The World* on Roblox.

---

## 📦 Files Included

- `ETW_MACRO_v1.0.1.ahk` – Main AutoHotkey macro script  
- `ETW_MACRO_v1.0.1.exe` – Compiled executable version  
- `Autoclicker.ahk` – Autoclicker module  
- `Gdip_all.ahk` – Screen capture library for AutoHotkey  
- `link.txt` – Personal configuration file required for macro setup  
- `version.txt` – Tracks current macro version  

---

## ⚙️ Features

- 🗺️ Auto Mega Maps  
- 🗳️ Map voting system (choose preferred map)  
- 🎁 Auto crate claiming at optimal timing  
- 💰 Auto sell when inventory is full  
- ⬆️ Upgrade system every 1.5 hours (future automation planned)  
- 🔁 Auto rejoin when crates are completed  

This macro ensures continuous grinding in-game with minimal input.

---

## 🚀 How to Run

### 1. Download Requirements
- Install **Tesseract OCR**:  
  https://github.com/UB-Mannheim/tesseract/wiki

### 2. Install AutoHotkey
- Required for `.ahk` scripts: https://www.autohotkey.com/

### 3. Setup
- Download this repository  
- Ensure Tesseract is installed and added to PATH  
- Configure `link.txt` with your system values  

### 4. Run
- Run `ETW_MACRO_v1.0.1.exe` (recommended)  
  **OR**
- Run `ETW_MACRO_v1.0.1.ahk` using AutoHotkey
