# 🍽️ ETW Macro Roblox

ETW Macro is an automation script for **Eat The World** on Roblox, built with **AutoHotkey v2** and **Tesseract OCR**.

---

## 📦 Requirements

Before using the macro, install the following:

### 1. Install AutoHotkey v2
Download and install:  [Here](https://www.autohotkey.com/v2/)

### 2. Install Tesseract OCR
Download and install:  [Here](https://github.com/UB-Mannheim/tesseract/wiki)
